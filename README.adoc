* Java 11

* https://vuejs.org/[Vue.js] for web client
* https://vuex.vuejs.org/[Vuex] for State management
* https://www.javascript.com/[JavaScript] for web client
* https://antdv.com/docs/vue/introduce/[Antdv] as design template
